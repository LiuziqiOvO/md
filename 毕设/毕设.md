

# 1 绪论

## 1.1 课题背景、目的与意义

随着数据中心对存储性能与效率的需求日益增长，传统的存储解决方案已经无法满足这些日益增长的需求。这种背景下，Zoned Namespace (ZNS) SSD作为一种新兴的存储技术，因其独特的数据管理方式和对存储效率的显著提升而受到学术界和工业界的广泛关注。ZNS SSD通过引入分区的概念来优化数据的写入过程，从而减少写入放大效应，提高存储设备的寿命和性能。本课题来源于实验室与浪潮等企业联合申请的“面向新型计算模式的分布式存储系统”重点专项，尝试利用ZNS SSD等新兴硬件重新设计存储系统。本课题作为此专项的先行性研究，准备基于多块实际的ZNS SSD盘做ZNS SSD的数据访问特征以及性能优化方面的研究。

本课题旨在深入分析基于Zoned Namespace (ZNS) SSD的数据访问特征，并通过研究针对这些特性的性能优化策略，探索提升基于ZNS SSD的存储系统的性能的有效方法。相较于现有在QEMU[1]、FEMU[2]、NVMeVirt[3]等模拟器上的工作，此项工作不仅能够为理解ZNS SSD在不同的真实应用场景下的性能表现提供理论基础，而且通过性能优化实验，能够为数据中心的存储系统设计提供实验参考，从而在实际应用中提升存储系统的效率和可靠性。此外，随着数据中心对存储解决方案效率和性能要求的不断提升，该方向的研究有望为存储技术进步和创新贡献力量，满足未来数据中心更为复杂和高效的存储需求。
